# Dart Fundamentals Assignment

This repository contains two tasks for demonstrating fundamental concepts in Dart programming, including list analysis, loops, conditionals, Object-Oriented Programming (OOP), exception handling, and asynchronous programming.

## Repository Structure
- `task1/number_analysis.dart`: Logic for analyzing a list of numbers. 
- `task2/calculator_app.dart`: An asynchronous calculator implementation with exception handling and OOP principles.
- `reflection.md`: Reflection on the assignment.
- `README.md`: Project overview.

## How to Run
If you have Dart installed, you can run the tasks from your terminal:

1.  To run the Number Analysis App:
    ```bash
    dart task1/number_analysis.dart
    ```
2.  To run the Async Calculator App:
    ```bash
    dart task2/calculator_app.dart
    ```

Alternatively, you can copy the code into [DartPad](https://dartpad.dev) to run it in your browser.
